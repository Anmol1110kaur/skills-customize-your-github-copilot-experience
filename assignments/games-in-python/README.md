

# 📘 Assignment: Hangman Game Challenge

## 🎯 Objective

Create a classic Hangman word-guessing game in Python! You will practice using strings, loops, conditionals, and random selection to build an interactive game where players guess letters to reveal a hidden word before running out of attempts.

## 📝 Tasks

### 🛠️ Task 1: Build the Hangman Game

#### Description
Write a Python program that lets a player guess letters to uncover a secret word. The game should keep track of correct and incorrect guesses, and end when the player either guesses the word or runs out of attempts.

#### Requirements
Completed program should:

- Randomly select a word from a predefined list
- Accept single-letter guesses from the player
- Show the current progress using underscores for unguessed letters (e.g., _ _ t h o n)
- Track and display the number of incorrect guesses remaining
- End the game when the word is fully guessed or attempts are used up
- Display a congratulatory message if the player wins, or a game over message if they lose

#### Description
Challenge yourself by adding extra features to your Hangman game!

#### Requirements
If you choose to extend your game, you could:

- Add a hint system that reveals a letter
- Allow the player to choose difficulty (number of attempts)
- Keep track of and display all guessed letters
- Add ASCII art for the hangman
